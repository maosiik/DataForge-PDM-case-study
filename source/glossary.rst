Glossary of Terms
=================

.. glossary::

   Check-in
      The process of uploading a modified file back to the PDM vault, creating a new version, and unlocking it for other users.

   Check-out
      The process of downloading a file from the PDM vault to a local workspace and locking it so no one else can edit it simultaneously.

   ERP
      Enterprise Resource Planning is a corporate software system (such as SAP) used to manage daily business activities, including manufacturing and supply chain. DataForge PDM often exports BOMs directly to ERP systems.

   In Review
      A lifecycle state where the file is locked and waiting for a manager's approval.

   Released
      A lifecycle state where the file is fully approved, locked against further changes, and ready for manufacturing. Only files in this state can be used for BOM exports.

   Work in Progress
      A lifecycle state indicating the file is currently being edited by an engineer. It is not ready for any exports.
