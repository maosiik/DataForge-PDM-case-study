Release Notes (Version 26.3)
============================

This update of DataForge PDM primarily brings faster preparation of manufacturing data and new integrations for your existing corporate systems.

New Features
------------
* **Bulk BOM Export:** Users can now download complete BOMs, including drawings, with a single click directly from the assembly's context menu.
* **REST API for Exports:** We have added a new endpoint for direct integration of BOM exports into your ERP systems.

Improvements and Bug Fixes
--------------------------
* Improved the clarity of error messages when attempting to export unapproved data.
* Added a new introductory guide explaining the file locking process for new users.
